# PeerSync-Rating-System
PeerSync is a C# desktop application that helps students find compatible teammates for academic projects. The system allows users to create profiles, discover potential teammates through a swipe-based interface, form project groups, and provide ratings based on collaboration experiences.

## Features

- User Registration and Login
- Student Profile Management
- Swipe-Based Peer Discovery
- Team Matching System
- Project Group Formation
- Peer Rating and Feedback
- User-Friendly Interface

## Technologies Used

- C#
- Windows Forms
- .NET Framework
- SQL Server

## Project Objectives

The main goal of PeerSync is to simplify the process of finding suitable project partners and improve collaboration among students through a structured matching and rating system.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/SajalMollick/PeerSync.git
```

2. Open the solution file (.sln) in Visual Studio.

3. Restore required packages.

4. Build and run the project.
